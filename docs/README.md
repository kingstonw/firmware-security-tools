# WatusiPanel V3 Documentation

```
docs/
├── guides/              # 项目规范和开发指南
├── rust/                # Rust开发相关文档
├── architecture/        # 系统架构和设计文档
```

### [rust](./rust/) - Rust开发
与Rust集成、构建和故障排除相关的所有文档。

### [architecture](./architecture/) - 架构设计
系统架构、组件设计和模块化结构文档。

### [migration](./migration/) - 迁移管理
代码重构、模块化迁移相关的详细记录。

### [features](./features/) - 功能文档
设备配置、OTA升级、日志处理等功能的实现文档。

### [broadcast](./broadcast/) - 广播功能
广播相关的功能文档。
